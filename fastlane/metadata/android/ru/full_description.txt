Просто приложение, которое позволяет вам изменить значение TTL (время жизни IP пакетов) по умолчанию. Основные возможности:

* Чтение/запись значения TTL
* Поддержка IPv6
* Применение последнего записанного значения после перезагрузки
* Виджеты для изменения TTL в одно нажатие

Приложение требует права суперпользователя для работы.
